# On the Theory of Elliptic Functions

English translation of the 1877 paper "Zur Theorie der elliptischen Functionen" by Ferdinand Georg Frobenius and Ludwig Stickelberger.

## About

This repository contains an English translation of the _Journal für die reine und angewandte Mathematik_ (Crelle's Journal), Vol. 83, pp. 175-179 (1877).

## Files

- `fs_determinant/zur_theorie_elliptischen_functionen.tex` - LaTeX source of the translation
- `fs_determinant/latex/zur_theorie_elliptischen_functionen.pdf` - Compiled PDF

## Original Paper

**Reference:** Frobenius, F. G., & Stickelberger, L. (1877). Zur Theorie der elliptischen Functionen. _Journal für die reine und angewandte Mathematik_, 83, 175-179.
